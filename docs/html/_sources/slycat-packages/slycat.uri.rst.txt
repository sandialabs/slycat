slycat.uri
==========

Provides server-side functionality for creating, parsing, and editing Uniform
Resource Identifiers (URIs) using an API that is based on the excellent
`URI.js <http://medialize.github.io/URI.js/>`_ library (which is available for
Slycat clients).
