Supported Media Formats

The Parameter Space model supports the retrieval and display of many different media formats. The display of media is enabled by this model to provide exploration and understanding that connects features in a scatterplot with simulation artifacts. Media in these formats may be included in the data table used to create the Parameter Space model. See the Source Data, CSV Format Data Table page for instructions on including the path to these artifacts.

Image Formats:

Common image formats, such as JPEG and PNG, are supported. Slycat relies on the user’s web browser to render images. Ultimately, the supported image formats are determined by the user’s Operating System and particular web browser being used.

Video File Formats:

Common video formats, such as MP4 and OGG, are supported. Slycat relies on the user’s web browser to render videos. Ultimately, the supported video formats are determined by the user’s Operating System and particular web browser being used.

3D Formats:

  • Stereolithography CAD (STL)

  • VTK Polygonal Data (VTP)

Document Formats:

  • Portable Document Format (PDF)
